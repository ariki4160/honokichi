## Honokichi
Honokichi is blog theme for [Hugo](http://hugo.spf13.com).  
This theme based on [Honoka](http://honokak.osaka/).

## Feature
### support
- posts
- pages
- tags
- sns buttons

### not support
- categories

## Settings
```
[params.social]
  twitter_id = "your twitter screen name"
  hatena = true
  github = "ota42y"
```
